To use the css file include the following between the <head> tags: 

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/Pingu1337/pinguLib@latest/pages/assets/css/pinguLib.css">



Copy-paste the <script></script> right before the closing of the </body> tag to enable extra features that require JavaScript. 

<script src="https://cdn.jsdelivr.net/gh/Pingu1337/pinguLib@latest/pages/assets/js/pinguScripts.js"></script>
